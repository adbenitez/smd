# Simple Manga Downloader

A command line program to automatically download(scrap) manga from manga web pages.

# Example
`./smd.py -s mangaall 'one piece'`

# Currently supported sites:
* tumangaonline
* ninemanga-en
* ninemanga-es
* ninemanga-ru
* ninemanga-de
* ninemanga-it
* ninemanga-br
* heavenmanga
* mangareader
* mangaall
